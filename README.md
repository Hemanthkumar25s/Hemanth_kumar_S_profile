# Hemanth Kumar S — Portfolio Website

A dark-themed personal portfolio built with Next.js, TypeScript, Tailwind CSS, and Framer Motion. Features a modern glassmorphism UI with smooth animations, showcasing projects, skills, and contact details for a student developer focused on web and mobile development.

> 🌐 [Live Site](https://hemanth-kumar-s-profile.vercel.app/) • 📧 hemanthkumar.s3125@gmail.com • 💼 [LinkedIn](https://linkedin.com/in/hemanthkumars25) • 🐙 [GitHub](https://github.com/Hemanthkumar25s)

Built with Next.js 15, TypeScript, Tailwind CSS, and Framer Motion. Features include a responsive layout, dark glassmorphism theme, smooth page transitions, typewriter effects, 3D tilt cards, and an interactive cursor pet companion.

Deployed via Vercel with continuous integration from the main branch.
